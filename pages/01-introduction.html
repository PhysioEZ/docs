Welcome to the CareSyncOS Docs

This documentation is the central source of truth for developers working on the CareSyncOS platform. Its purpose is to provide all the necessary information to understand the architecture, set up a local development environment, and contribute effectively and consistently to the project.

Whether you are a new developer joining the team or an experienced contributor looking for specific information, this site is your primary resource.

How to Use These Docs

The documentation is designed to be easy to navigate:

  • Main Navigation (Left Sidebar): The primary navigation on the left groups documentation into logical categories like "Local Development," "Database Schema," and "Backend Architecture." Use this to explore the main topics.
  • On This Page (Right Sidebar): For longer articles, a table of contents will automatically appear on the right. Use this to quickly jump to a specific section on the current page.
  • Search: The search bar at the top of the sidebar allows you to quickly find pages by keyword.

What You'll Find Inside

This documentation covers the entire system, from high-level concepts to low-level implementation details. You will find detailed guides on the project's architecture, database schema, key business logic workflows, and instructions for setting up a local development environment.
