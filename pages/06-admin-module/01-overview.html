Admin Module Overview

The Admin Module is the central control panel for CareSyncOS. It is designed for users with the 'admin' role and provides the highest level of access to system configuration, user management, and operational oversight.

From this module, administrators can manage branches, onboard new employees, set roles and permissions, monitor system-wide activity through audit logs, and manage financial data like expenses and budgets. It is the heart of the system's administrative functions.

Module Structure

The documentation for the admin module is broken down into the following key areas, mirroring its directory structure:

  • Views: Detailed explanations of each page and interface available to the administrator, such as the dashboard, employee management page, and branch settings.
  • API Endpoints: Documentation for the backend scripts located in /admin/api/. These handle the core logic for creating, updating, and deleting data.
  • Client-Side Scripts: A guide to the JavaScript files in /admin/js/, which power the dynamic and interactive features of the admin panel.
Prerequisite: All pages within this module are protected by common/auth.php and require an active session with the 'admin' role.
