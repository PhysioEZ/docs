Local Development Setup

This guide provides step-by-step instructions for setting up the CareSyncOS project on your local machine for development and testing. Following these steps will ensure you have a working environment that mirrors the production setup.

Prerequisites

Before you begin, ensure you have the following software installed on your system:

  • XAMPP: A local server environment that includes Apache, MariaDB (or MySQL), and PHP. The application requires PHP 8.0 or higher. You can download it from the official XAMPP website.
  • Git: For cloning the project repository.
  • Code Editor: A text editor or IDE for code editing, such as Visual Studio Code.

Installation Steps

Follow these steps in order to get the application running.

Step 1: Place the Project Folder

Place the entire OS project directory inside your XAMPP's web root folder. This is typically located at: 

C:\xampp\htdocs\

After this step, your project path should be C:\xampp\htdocs\OS\.

Step 2: Set Up the Database

  1. Start the Apache and MySQL services from your XAMPP Control Panel.
  2. Open your web browser and navigate to http://localhost/phpmyadmin.
  3. Create a new database. For consistency, you can name it caresyncos.
  4. Select the newly created database and go to the "Import" tab.
  5. Click "Choose File" and select the provided SQL dump file (e.g., database.sql) from the project repository.
  6. Toggle Foreign Key check off to ensure data integrity.
  7. Click "Go" at the bottom of the page to import the schema and data.

Step 3: Configure the Application

The database connection details and other system keys are stored in common/config.php. You will need to update this file with your local database credentials.

Open the file /OS/common/config.php and set the database credentials you created in the previous step.

<?php
// common/config.php

// --- Database Configuration ---
define('DB_HOST', 'localhost');
define('DB_USER', 'root'); // Your database username
define('DB_PASS', '');     // Your database password
define('DB_NAME', 'caresync_os'); // The database name you created

// ... other configuration ...
?>

Verification

Once you have completed the steps above, you can verify that the application is running correctly.

Open your browser and navigate to the following URL: 

http://localhost/OS/

If the setup was successful, you should be automatically redirected to the main login page. You can now use the default credentials to log in and test the system.
